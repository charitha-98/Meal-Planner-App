# 🥗 Meal Planner App

A modern, user-friendly **Flutter** application designed to simplify meal planning and healthy eating. This app allows users to discover delicious recipes, track their nutritional intake, and organize their weekly meals with ease.

---

## ✨ Key Features

* **🔍 Smart Recipe Search:** Find recipes based on ingredients, meal types (Breakfast, Lunch, Dinner), or dietary preferences.
* **📅 Weekly Meal Planner:** Organize your meals for the entire week to save time and reduce food waste.
* **📊 Nutritional Insights:** View detailed information for each recipe, including calories, proteins, fats, and carbs.
* **🛒 Auto-Generated Shopping List:** Create a grocery list automatically based on your planned meals.
* **❤️ Favorites:** Save your favorite recipes for quick access anytime.
* **🌓 Dark & Light Mode:** Support for both themes for a comfortable user experience.

---

## 🛠️ Tech Stack

* **Framework:** [Flutter](https://flutter.dev/) (v3.x)
* **Language:** [Dart](https://dart.dev/)
* **State Management:** [Provider](https://pub.dev/packages/provider)
* **Data Source:** REST API Integration / Local JSON
* **Storage:** Shared Preferences / SQLite

## 🚀 Installation & Setup

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/charitha-98/Meal-Planner-App.git](https://github.com/charitha-98/Meal-Planner-App.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Meal-Planner-App
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
